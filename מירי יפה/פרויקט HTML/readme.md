Sweet Petite הוא אתר תדמית וקטלוג דיגיטלי לקונדיטוריית בוטיק המתמחה בפטיפורים וקינוחים אישיים.מטרת האתר היא להציג את המותג, הקינוחים והשירותים בצורה אסתטית, ברורה ונעימה למשתמש, תוך דגש על עיצוב מודרני, חוויית משתמש ונגישות.
האתר מהווה בסיס עיצובי ומבני לפיתוח עתידי, וכולל הצגת מוצרים, עמודי תוכן, גלריה, חוות דעת ויצירת קשר.

### 2. ארכיטקטורת עיצוב (Styling Architecture)
העיצוב מפוצל ל קבצי CSS כדי לשמור על סדר:
* **`A.css`** - הגדרות גלובליות, פונטים, Header ו-Footer (משותף לכל הדפים).
* **`A.css`** - עיצוב ייחודי לדף הבית הכולל את האנימציות הראשיות.
* **`gallery.css`** - עיצוב הגריד והכרטיסיות של הקטלוג.
* **קבצי עיצוב ייעודיים** נוספים לשאר דפי התוכן.

## 🛠️ דגשים טכניים וביצועים

### חלוקה סמנטית (Semantic HTML)
בניתי את האתר בחלוקה קשיחה לתגיות סמנטיות:
* **`<header>`**: תפריט ניווט קבוע עם אפקטים של Hover.
* **`<main>`**: התוכן הדינמי המשתנה בין דף לדף.
* **`<footer>`**: אזור יצירת קשר וניתוב מהיר (Quick Links).
